修改Python的pip安装源

Date Mon 23 November 2015 By Alex Category Python技术. Tags pip

在 python 中用pip安装依赖包时默认会从 https://pypi.python.org/simple/这个地址取下载，但是这个网站经常出现下载不稳定、访问速度非常慢的情况，这时，我们可以修改配置用国内厂商提供的镜像地址来替换，一个是豆瓣的镜像，另一个是中国科学技术大学的。地址分别是：

  1. 豆瓣：http://pypi.douban.com/
  2. 中国科学技术大学 http://pypi.mirrors.ustc.edu.cn/simple/

在用pip安装第三方包的时候，有两种方式使用我们自己指定的镜像源

手动指定

在 pip 命令后面直接指定，例如：

pip -i http://pypi.douban.com/simple install Flask -- trusted-host pypi.douban.com

不过这种方式在每次安装时都要手动指定，很麻烦，所以我们可以使用第二种方式

写入配置

把镜像地址写到配置文件中，如果是Widnows系统，那么可以在当前用户目录下面创建 pip 文件夹，再创建pip.ini的文件，写入内容：

[global]
trusted-host =  pypi.douban.com
index-url = http://pypi.douban.com/simple

如果是Mac或者Linux用户，则在当前用户下创建.pip文件夹

mkdir ~/.pip

然后在该目录下创建pip.conf文件，一样输入内容：

[global]
trusted-host =  pypi.douban.com
index-url = http://pypi.douban.com/simple
