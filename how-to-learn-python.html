自学Python编程怎么学才不那么孤独

Date Tue 24 October 2017 By Alex Category Python技术. Tags python

Python 是当下最流行的编程语言，甚至高盛的一个调查表示 Python 超过汉语成为未来最重要技能，虽然有点耸人听闻，随着人工智能渗入到我们生活中的方方面面，以后编程就跟现在汽车一样普及，成为一项基本技能。

Python 上手简单，拥有简洁的语法。用途广泛，人工智能、机器学习、数据分析、数据挖掘、网络爬虫， Web 开发等。Python 还可以做运维开发、自动化测试、金融量化交易、渗透工具，总之，学会了 Python，赚钱的门路很多。

那么 Python 到底好不好学？自学能力强的朋友，完全可以自己在网络上找资源自学，但是可能会走很多弯路，一个人自学会很孤独，很容易半途而废。而自学感觉吃力的朋友反而会想办法找有经验的人寻求帮助或者进入某个圈子，最后少走很多弯路，学起来也更快。

经过很多朋友的催促，我开通了知识星球圈子（原来的小密圈），给想学 Python 但又缺乏自制力，或者有能力但是感觉一个人孤军奋战很困难的朋友们提供一个成长的途径。

在这里没有闲聊、没有灌水、没有乱七八糟的广告，就是一个纯粹学习的地方，我会在这里不定期与大家分享一些公众号之外的东西，包括但不限于技术、学习方法、职场、个人成长方面的话题。你还可以向我提问，每天我都会抽出时间集中回答一些问题。同时，还会有一些新奇玩法，比如组团学习，在这里，你的学习目标将被大家一起监督。更重要的，你可以在这里与志同道合的人一起学习、交流思想、共同进步。
